# mq-shift
## Install Nodejs
https://github.com/nodesource/distributions
Example
```
curl -fsSL https://deb.nodesource.com/setup_16.x | sudo -E bash -
sudo apt-get install -y nodejs
```

## Create the app
```
npx create-react-app app
cd app
npm start
npm install react-router-dom react-select
npm start
```
